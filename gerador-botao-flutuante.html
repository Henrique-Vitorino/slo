Crie um botão flutuante de WhatsApp — ou um menu com vários canais — para o seu site. Personalize cor, ícone e posição e copie o código pronto. Sem plugin, sem cadastro.

Perguntas Frequentes

Tire suas dúvidas sobre o botão flutuante.

Preciso instalar algum plugin?expand_more
Não. O código gerado é HTML e CSS puros, sem bibliotecas nem scripts externos. É só colar antes do </body> do seu site — menos peso e menos risco que um plugin.
Funciona em WordPress, Shopify e Nuvemshop?expand_more
Sim, funciona em qualquer plataforma que permita inserir HTML no tema. No nosso guia do botão flutuante mostramos onde colar o código em cada plataforma.
Posso usar sem ser para WhatsApp?expand_more
Pode. Além de WhatsApp, o botão aceita Instagram, Facebook, Telegram, telefone, e-mail e qualquer link — como uma página de promoção, catálogo ou agendamento. Escolha o ícone e a cor que combinam com sua marca.
O botão deixa meu site lento?expand_more
Não. O código tem poucos KB, não carrega nada de servidores externos e usa apenas CSS para as animações. É uma das formas mais leves de adicionar um botão de contato.
